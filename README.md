# MusicPad
A macropad designed to make music control feel mechanical. Simply plug into your computer and feel like a DJ!

<img width="509" height="423" alt="image" src="https://github.com/user-attachments/assets/a946bcc1-6810-4dd5-b532-2df999d79462" />

Features:
- Keyboard switches for play/pause, skip forward/back and shuffle
- Rotary volume control with integrated mute switch
- 0.91" OLED to display current song
